이 문서는 야구 시뮬레이터에서 사용되는 **모든 스탯 항목**을 정리한 레퍼런스입니다.
새 스탯이 추가되거나 구현 상태가 변경될 때 이 문서를 업데이트합니다.

---

## 범례

| 기호 | 의미 |
|------|------|
| ✅ | 구현됨 (타입 정의 + 엔진 반영) |
| 📋 | 미구현 (계획됨) |
| 🔢 | 파생값 (다른 스탯에서 계산) |
| `[능력치]` | `PlayerStats`에 정의된 능력치 스탯 |
| `[성적]` | 경기/시즌 결과로 집계되는 성적 스탯 |

---

## 1. 능력치 스탯 (PlayerStats)

선수가 보유한 고유 능력치. 경기 중 판정에 직접 사용됩니다.
`src/lib/baseball/types/player.ts`의 `PlayerStats` 인터페이스에 정의됩니다.

### 투수 능력치

| 스탯 | 필드명 | 범위 | 상태 | 설명 |
|------|--------|------|------|------|
| 구위 | `ball_power` | 30~100 | ✅ | 피타구 억제력. 타구 품질에 영향 |
| 제구 | `ball_control` | 30~100 | ✅ | 의도한 코스에 던지는 능력 |
| 변화 | `ball_break` | 30~100 | ✅ | 변화구 구사력. 스윙 미스 유도 |
| 구속 | `ball_speed` | 30~100 | ✅ | 투구 속도. 도루 판정에도 사용 |
| 스태미나 | `stamina` | 0~100 | ✅ | 투구 가능 체력. 소진 시 교체 |

> **구종별 스탯** (`PitchTypeData`): 각 구종마다 `ball_power`, `ball_control`, `ball_break`, `ball_speed`, `weight(구사비율)` 보유. `PlayerStats`의 투수 능력치는 구종별 가중 평균으로 산출.

### 야수/공통 능력치

| 스탯 | 필드명 | 범위 | 상태 | 설명 |
|------|--------|------|------|------|
| 컨택 | `contact` | 30~100 | ✅ | 스윙 정확도. 안타 확률에 영향 |
| 파워 | `power` | 30~100 | ✅ | 타구 비거리. 장타/홈런 확률에 영향 |
| 수비 | `defence` | 30~100 | ✅ | 수비 능력. 도루 성공률 판정에도 사용 |
| 송구 | `throw` | 30~100 | ✅ | 송구 정확도/속도. 도루/견제 판정에 사용 |
| 주루 | `running` | 30~100 | ✅ | 주루 속도. 도루 시도/성공률에 사용 |
| 스태미나 | `stamina` | 0~100 | ✅ | 공통. 투수는 투구 소모, 타자는 미사용(현재) |

### 기타 선수 속성

| 속성 | 필드명 | 상태 | 설명 |
|------|--------|------|------|
| 스트라이크 존 하단 | `zone_bottom` | ✅ | 타자 체형 반영 (m 단위) |
| 스트라이크 존 상단 | `zone_top` | ✅ | 타자 체형 반영 (m 단위) |

## 2. 타자 성적 스탯

경기 결과로 집계되는 타격 기록. `GameEvent[]`를 소비하여 계산합니다.

### 기본 집계 스탯

| 스탯 | 필드명 | 상태 | 집계 조건 |
|------|--------|------|-----------|
| 타수 | `AB` | 📋 | single/double/triple/home_run/out/strikeout 시 +1 (walk/HBP 제외) |
| 안타 | `H` | 📋 | single/double/triple/home_run 시 +1 |
| 1루타 | `1B` | 📋 | single 시 +1 |
| 2루타 | `2B` | 📋 | double 시 +1 |
| 3루타 | `3B` | 📋 | triple 시 +1 |
| 홈런 | `HR` | 📋 | home_run 시 +1 |
| 볼넷 | `BB` | 📋 | walk/hit_by_pitch 시 +1 |
| 삼진 | `SO` | 📋 | strikeout 시 +1 |
| 타점 | `RBI` | 📋 | 타격 결과로 홈인한 주자 수 + (홈런 시 +1) |
| 도루 | `SB` | 📋 | steal_result.success === true 시 +1 |
| 도루실패 | `CS` | 📋 | caught_stealing / steal_result.success === false 시 +1 |

### 파생 스탯 (계산값)

| 스탯 | 필드명 | 상태 | 계산식 |
|------|--------|------|--------|
| 타율 | `AVG` | 🔢📋 | H / AB |
| 출루율 | `OBP` | 🔢📋 | (H + BB) / (AB + BB) |
| 장타율 | `SLG` | 🔢📋 | (1B + 2×2B + 3×3B + 4×HR) / AB |
| OPS | `OPS` | 🔢📋 | OBP + SLG |

### 미구현 / 향후 추가 예정

| 스탯 | 상태 | 비고 |
|------|------|------|
| 사구(HBP) | 📋 | BB와 별도 집계 필요 시 |
| 희생번트 | 📋 | 번트 stub 해제 후 추가 |
| 희생플라이 | 📋 | SF 판정 로직 필요 |
| 출장 경기 수 | 📋 | 시즌 스탯 피처에서 처리 |
| 연속 안타 | 📋 | 시즌 스탯 피처에서 처리 |

## 3. 투수 성적 스탯

경기 결과로 집계되는 투구 기록.

### 기본 집계 스탯

| 스탯 | 필드명 | 상태 | 집계 조건 |
|------|--------|------|-----------|
| 이닝 | `IP` | 📋 | 아웃 1개 = +0.1, 0.3 → 1.0 올림 처리 |
| 피안타 | `H` | 📋 | 상대 타자 single/double/triple/home_run 시 +1 |
| 자책점 | `ER` | 📋 | 실점 발생 시 +1 (수비 에러 미구현 → 모든 실점 = ER) |
| 볼넷허용 | `BB` | 📋 | walk/hit_by_pitch 허용 시 +1 |
| 탈삼진 | `SO` | 📋 | 삼진 시 +1 |
| 승 | `W` | 📋 | 팀 승리 + 리드를 취한 시점의 투수 |
| 패 | `L` | 📋 | 팀 패배 + 결승점 허용 투수 |
| 세이브 | `SV` | 📋 | 승리팀 마지막 투수 + 세이브 조건 충족 |

### 파생 스탯 (계산값)

| 스탯 | 필드명 | 상태 | 계산식 |
|------|--------|------|--------|
| 방어율 | `ERA` | 🔢📋 | ER × 9 / IP |
| WHIP | `WHIP` | 🔢📋 | (BB + H) / IP |

### 세이브 조건 (Simulator 기준)
```
세이브 기준값 SaveConditionValue = 2
조건: 점수차 - (등판 시 주자 수 + 1) < (2 + 남은 이닝 수)
```

### 미구현 / 향후 추가 예정

| 스탯 | 상태 | 비고 |
|------|------|------|
| 비자책점(R) | 📋 | 수비 에러 구현 후 ER과 분리 |
| 완투(CG) | 📋 | 시즌 스탯 피처에서 처리 |
| 완봉(SHO) | 📋 | 시즌 스탯 피처에서 처리 |
| 홀드(HLD) | 📋 | 기획 필요 |
| 블론세이브(BS) | 📋 | 기획 필요 |

## 5. 능력치 스탯 vs 성적 스탯 구분 기준

| 구분 | 정의 | 예시 |
|------|------|------|
| **능력치** | 선수가 보유한 고유 잠재력. 경기 판정 공식에 직접 대입. 게임 외적으로 관리. | contact, power, ball_speed |
| **성적** | 경기/시즌 결과를 누적한 기록. `GameEvent[]`를 소비하여 산출. | H, HR, ERA, AVG |
